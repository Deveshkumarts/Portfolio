## 🚀 About the Project

This is a highly modern, futuristic cybersecurity-themed portfolio website designed to showcase projects, skills, and experience. Built with a hacker mindset, the UI integrates glassmorphism, dynamic glowing neon outlines, glitch animations, and an interactive digital matrix rain background to provide a fully immersive terminal aesthetic.

## ✨ Features

- **Cyberpunk Aesthetic:** Deep dark slate backgrounds mixed with neon cyan, purple, and green.
- **Matrix Rain Canvas:** A dynamic background element simulating classic terminal data streams.
- **Glitch & Glow Effects:** State-of-the-art CSS animations providing true cyberpunk vibrancy.
- **Glassmorphism Panels:** Frosted-glass components enhancing depth and maintaining a modern interface.
- **Fully Responsive:** Optimized for desktops, tablets, and encrypted mobile devices.
- **Secure Contact Form:** A visually appealing "encrypted communication" form.
- **Performance Optimized:** Built with Next.js App Router for high-speed delivery.

## ⚙️ Installation

To initialize this portfolio on your local machine, run the following sequence:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Devesh/Cyber-Portfolio.git
   ```

2. **Navigate to the directory:**
   ```bash
   cd Cyber-Portfolio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Initialize the development server:**
   ```bash
   npm run dev
   ```

5. **Access the terminal:**
   Open [http://localhost:3000](http://localhost:3000) with your browser to witness the result.

## 🌐 Deployment

This application is structurally engineered for instantaneous deployment via **Vercel**. 

1. Push your code to your GitHub repository.
2. Sign in to your [Vercel](https://vercel.com/) dashboard.
3. Click "Add New Project" and import your GitHub repository.
4. Leave all build settings to default (`npm run build`).
5. Complete deployment. Your site is now live on the world wide web.
